# ytplay
>


## Install
Using `npm`:
```
$ npm install --save ytplay
```

Using `bower`:
```
$ bower install --save ytplay
```

`ytplay` can be used as a CommonJS module, AMD (with Require.js) or as a plain old Javacript global.
